## 🛠️ Installation & Setup

### 🐳 Docker Installation (Recommended)

<details open>
<summary><strong>🔥 Quick Start with Docker</strong></summary>
<br>

```bash
# Pull and run the Docker container
docker pull snowball01/adultcolony-api:latest
docker run -p 2025:2025 snowball01/adultcolony-api:latest
```

<div align="center">
  <img src="https://img.shields.io/badge/Docker-2496ED?style=for-the-badge&logo=docker&logoColor=white" alt="Docker"/>
  <p><em>🎉 That's it! Your API is now running at http://localhost:3000</em></p>
</div>

</details>

### 🔧 Manual Installation

<details>
<summary><strong>📦 Manual Setup Steps</strong></summary>
<br>

```bash
# Clone the repository
git clone https://github.com/Snowball-01/adultcolony-api.git

# Navigate to the project directory
cd adultcolony-api

# Install dependencies
npm install

# Build the project
npm run build

# Start the server
npm run start:prod
```
